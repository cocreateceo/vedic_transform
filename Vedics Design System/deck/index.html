10X Vedic Transform combines ancient Vedic wisdom — Brahma Muhurta, Pranayama, Sandhya, Nidra — with a modern habit-tracking app. Users commit to a 48-day Mandala, complete 11 daily practices spanning body, mind, and spirit, and earn karma points + streak shields along the way.

Polish · the small details

Every interaction considered

⌘K

Command palette

Press Cmd-K (or Ctrl-K) from anywhere to jump to any of 18 destinations — including deep-links into Sessions sub-tabs.

🎨

Three themes

Light (default), Sattva (warm evening), Dark (high-contrast night). Tokens defined as CSS variables, toggleable via header sun icon, persists to localStorage.

📱

Mobile bottom nav + More sheet

Below 1024px: 4 primary tabs + a More button that opens a 5-column grid sheet of the 12 long-tail tools.

🛡

Streak shields

Visual badge inside StreakCounter when shields {">"} 0 — protects against one missed day. Phase reflection card fires on day 8/15/22/31/41.
